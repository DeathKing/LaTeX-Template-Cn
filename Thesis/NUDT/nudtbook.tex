\chapter{使用说明}
    由于各种原因，模版作者不可能提供详细的手册和示例。特别是对于~\TeX{}\slash\LaTeX\ 系统的安装调试，CJK、CCT
软件的配置，添加中文字体的方法，以及章节层次、公式、插图、表格等~\LaTeX\ 基本元素的
说明和使用方法，请用户自行参考有关书籍资料。其中一些有代表性的文献，已在“参考文献表”中给出。以下的叙述，
假定模版用户具备基本的~\TeX{}\slash\LaTeX\ 知识，可以熟练地使用系统标准类（如：article, book, cctart, cctbook 等）
撰写科技论文。
\section{所需软件环境}
    作者开发时所使用的软件环境如下：
\begin{quote}
  Windows 2003 + \CTeX{} 2.4.0 + CCT 0.6.1
\end{quote}
理论上讲，无论在何种操作系统下，只要
\begin{enumerate}
  \item 正确安装了一种~\TeX{}\slash\LaTeX\ 发行版（如：\TeX{}Live、Mik\TeX{}、te\TeX{}等）；
  \item 正确配置了~CJK；
  \item 正确配置了新版的~CCT（建议使用~CCT 0.6.1 以上版本）；
  \item 正确配置了有关字体（英文字体通常不需要额外配置，中文字体至少应包括黑体、宋体、仿宋体三种，
        其它字体是否需要安装，视论文内容而定）
\end{enumerate}
都应该可以正常使用该模版。本文的源文件可以用于测试用户的计算机是否具备上述条件。\par
    对于使用~MS Windows 操作系统的用户，推荐使用~\CTeX{} 套装系列软件，或者类似~
\TeX{}Live + \CTeX{}-CS + \CTeX{}-Fonts 这样的组合，可以大大简化安装配置的工作量。

\section{使用方法}
    \begin{enumerate}
      \item 将~nudtbook.cls、declaration.tex 及~chinesebst.bst 放在论文主文件所在目录下，
            或者~\TeX{}\slash\LaTeX\ 系统可以找到的目录下
      \item 在论文的导言区使用如下声明
        \begin{verbatim}
          \documentclass[phd]{nudtbook} %适用于博士学位论文
          或
          \documentclass[master]{nudtbook} %适用于硕士学位论文
        \end{verbatim}
        由于“master”为默认选项，所以硕士论文还可以更简单的使用如下声明
        \begin{verbatim}
          \documentclass{nudtbook} %适用于硕士学位论文
        \end{verbatim}
      \item 以下与使用~cctbook 类撰写论文的方法相同，具体的设置请参考本文的源文件。
    \end{enumerate}

\section{已知错误}
\begin{itemize}
  \item 未经官方认可。
  \item 目录部分的页码形式与样本文件不符
  \item 由于使用的是~CCT，所有类似“宋体粗体”这样的要求，都被黑体所代替（中文中似乎没有加粗的概念）
  \item 日期需要手动输入
  \item 题目过长时（比如超过~25 字），个别地方的显示需要手工调节（比如“独创性声明”）
  \item 如果封面上“专业方向”过长，可直接修改~nudtbook.cls 中的相应设置。
  \item 参考文献的列表尚不能完全自动完成，需要手动修改 .bbl 文件。主要的问题在于，中外文对参考文献
        格式的要求不同，比如“edition”这样的词不能自动转换。因此建议论文主体定稿后，再去修改 .bbl
        文件，并且修改后及时备份。
  \item 没有测试与其它宏包的兼容性。
  \item 模版只是完成了“国防科技大学撰写硕士、博士学位论文的规定”中对学位论文的格式要求，
        这些要求是最基本的。每个学科都可能会有对论文格式的特殊要求，但是这些并不包括在该模版中，
        用户可以通过使用其它宏包，或者自己编写代码来实现所需要的功能。
\end{itemize}
